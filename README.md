# 男のモテ飯ガイド

全編無料公開の漫画サイト。スワイプで漫画を読み進める。

## 機能

- ✅ **スワイプ操作** - 左右スワイプで前後のページに移動
- ✅ **キーボード操作** - 矢印キーでも操作可能
- ✅ **ページネーション** - 現在のページ位置を表示
- ✅ **進捗バー** - 読み進める状況を可視化
- ✅ **レスポンシブ** - モバイル・タブレット・デスクトップ対応
- ✅ **ページ保存** - 最後に見たページを記憶

## ファイル構成

```
motei-meshi-guide/
├── index.html        # メインページ
├── style.css         # スタイルシート
├── script.js         # ビューアーロジック
├── images/           # 漫画画像
│   ├── 1.png
│   ├── 2.png
│   ├── ...
│   └── 6.png
└── README.md         # このファイル
```

## 操作方法

### スワイプ（モバイル）
- **左スワイプ** → 次のページ
- **右スワイプ** → 前のページ

### ボタン
- **‹ ボタン** → 前のページ
- **› ボタン** → 次のページ

### キーボード
- **← キー** → 前のページ
- **→ キー** → 次のページ

## ローカル実行

```bash
# Python 3 を使用
python3 -m http.server 8000

# または Node.js
npx http-server
```

ブラウザで `http://localhost:8000` にアクセス

## デプロイ

### GitHub Pages
1. GitHub にリポジトリを作成
2. `main` ブランチにプッシュ
3. Settings → Pages → Deploy from branch を選択

### Manus ホスティング
Manus インフラでホスティング可能
